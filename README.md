# Mini Library Project

This project is a simple Java application that simulates a basic library system.

## Features

- Create and manage books
- Track book availability
- Manage readers
- Search books by title
- Count available books

## Classes

- **Book** → represents a book (title, author, pages, availability)
- **Reader** → represents a user of the library
- **Library** → manages books and operations
- **Main** → runs and tests the application

## Example

The program demonstrates:
- Borrowing and returning books
- Displaying available books
- Finding books by title
- Counting available books
